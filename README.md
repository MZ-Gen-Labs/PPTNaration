# PPTGenVoice2

PPTGenVoice2 は、PowerPoint のスライドノート（スピーカーノート）に記載されたテキストから音声を自動生成し、スライドに直接挿入することができる Windows 向け PowerPoint アドインです。

## ✨ 主な機能 (Features)

- **ノートから音声生成**: 各スライドのノート部分のテキストを読み取り、高品質な音声を生成します。
- **スライドへの自動挿入**: 生成された音声ファイル（メディア）を、対象のスライドに自動で配置します。
- **ワンクリック・インストーラー**: 複雑なアドイン設定は不要。専用のインストーラー（.exe）で誰でも数秒で導入できます。
- **クリーンなライセンス**: すべてのコードがクリーンに実装されており、ビジネスでもプライベートでも自由にご利用いただけます。

## 💻 動作環境 (Requirements)

- **OS**: Windows 10 / Windows 11
- **ソフトウェア**: Microsoft PowerPoint (Office 365, 2019, 2021 等)
- ※ **注意事項**: 本アドインは Windows の OS 機能（ファイルシステム操作等）を使用しているため、**Mac版の PowerPoint には対応しておりません**。

## 🚀 インストール方法 (Installation)

ご自身の環境に合わせて、以下の2つの方法からお選びいただけます。

### 1. インストーラーによる導入（推奨・最も簡単）
1. 本リポジトリの [Releases](../../releases) ページにアクセスします。
2. 最新バージョンの `PPTGenVoice2_Setup_vX.X.X.exe` をダウンロードします。
3. **PowerPoint を完全に終了した状態**で、ダウンロードした `.exe` ファイルを実行します。
4. インストールが完了し、PowerPoint を起動すると専用のリボン（メニュー）が追加されます。

### 2. 手動インストール（アドイン機能からの追加）
インストーラーを使わず、ご自身で任意の場所に保存して有効化する方法です。

1. 本リポジトリから `PPTGenVoice2.ppam` をダウンロードし、PC内の任意のフォルダに保存します。
2. PowerPoint を起動します。
3. キーボードの **`Alt` → `T` → `I`** を順番に押して、「アドイン」ダイアログを直接開きます。
   （または、リボンの「開発」タブ > 「PowerPoint アドイン」をクリックします）
4. 「新規追加」ボタンをクリックし、手順1で保存した `PPTGenVoice2.ppam` を選択して「閉じる」を押します。

💡 **Tip:** 手動インストールの場合は、この「アドイン」ダイアログ（`Alt` → `T` → `I`）から、いつでもアドインのチェックを外して無効化したり、一覧から削除したりすることが可能です。

## 📖 使い方 (Usage)

1. PowerPoint でプレゼンテーションファイルを開きます。
2. 音声を付けたいスライドの **ノート（スピーカーノート）** に、読み上げさせたいテキストを入力します。
3. PowerPoint 上部に追加された本アドインの専用リボンタブを開きます。
4. **「音声生成（または該当の実行ボタン）」** をクリックします。
5. 処理が完了すると、スライド内に音声アイコンが挿入され、プレゼンテーション実行時に再生されるようになります。

---

## 🗑 アンインストール方法 (Uninstallation)

インストーラー（.exe）を使用して導入した場合は、Windows 標準の機能を使ってシステムを汚すことなく綺麗に削除できます。

1. **PowerPoint を終了**します。
2. Windows の「設定」>「アプリ」>「インストールされているアプリ」を開きます。
3. アプリ一覧から **`PPTGenVoice2`** を検索し、「アンインストール」をクリックします。
※ 手動インストールした場合は、保存した `.ppam` ファイルをご自身で削除してください。
